VQA Organization

Task

Given an image and a natural language question about the image,building a system that can answer that question about the image. Image
Question What is in the image ?
Expected answers Boats, Water etc

Applications

  1. To help blind and visually-impaired users.
  2. To provide information about an image on the Web or any social media.
  3. To integrate VQA into image retrieval systems.

Approach

    In general, we can outline the approaches in VQA as follows:
  1. Extract features from the question.
  2. Extract features from the image.
  3. Combine the features to generate an answer.
  1. In the case of image features, CNNs pre-trained on VGGNet was used.
  2. Question and image encodings are combined via element wise multiplication.
  3. Regarding the generation of the answer, we modeled the problem as a classification task.
Neural Network Model
  1. A multi-layer perceptron (MLP) neural network classifier with three hidden layers and 1000 hidden units (dropout 0.5) in each layer with tanh non-linearity.
  2. An LSTM model followed by a softmax layer to generate the answer.

Other Possible Approaches

Bayesian Approach
The idea behind bayesian approaches is to model co-occurrence statistics of both the question and the image features, as a way of inferring relationships between questions and images. It models the probability of image features given the question features and the type of the answer.
